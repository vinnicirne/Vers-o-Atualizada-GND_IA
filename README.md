# GDN_IA - Creator Suite

Bem-vindo ao GDN_IA, uma plataforma SaaS de Inteligência Artificial Generativa.

View your app in AI Studio: https://ai.studio/apps/drive/1o3ff8EzlWEDI0N6QP_6AX2GGNA59YTyV

## Instalação e Execução

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`

2. Configure a API Key do Gemini no arquivo `.env.local` (ou renomeie `.env.example`):
   `GEMINI_API_KEY=sua_chave_aqui`

3. Inicie a aplicação:
   `npm run dev`

## Manutenção do Projeto

Para manter o projeto limpo e remover arquivos obsoletos (Lixo Digital) identificados nas análises de código, execute:

`npm run cleanup`

Isso removerá arquivos antigos como `services/gemini.ts`, `components/auth/Login.tsx` e outros componentes não utilizados.
